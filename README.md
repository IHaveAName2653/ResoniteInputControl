# ResoniteInputControl

A [Resonite](https://resonite.com/) mod that allows you to use Variables to toggle whether a controller action (i.e. Jump, Move and Turn) continue to function based on a variable in your User Root.

Current Variables:
* User/InputControl.Left.Move
* User/InputControl.Left.Turn
* User/InputControl.Left.Jump
* User/InputControl.Right.Move
* User/InputControl.Right.Turn
* User/InputControl.Right.Jump

## Installation (Manual)
1. Install [ResoniteModLoader](https://github.com/resonite-modding-group/ResoniteModLoader).
2. Place [ResoniteInputControl.dll](https://github.com/IHaveAName2653/ResoniteInputControl/releases/latest/download/ResoniteInputControl.dll) into your `rml_mods` folder. This folder should be at `C:\Program Files (x86)\Steam\steamapps\common\Resonite\rml_mods` for a default install. You can create it if it's missing, or if you launch the game once with ResoniteModLoader installed it will create the folder for you.
3. Start the game. If you want to verify that the mod is working you can check your Resonite logs.
